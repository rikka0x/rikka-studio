# Product

## Vision

Make prompt experiments easy to run, compare, and document.

## Mission

Rikka Studio gives creators and developers a clean place to test prompts, review model responses, and export reusable notes without exposing provider credentials in the browser.

## Users

- creators testing content workflows
- developers checking model behavior
- operators documenting repeatable prompts
- teams comparing prompt variants before moving them into production

## Core use cases

1. Draft a system prompt and user prompt.
2. Run a demo response or connected model response.
3. Review the generated output and run metadata.
4. Export the prompt, response, and settings as Markdown.

## Non-goals for v0.1

- no login
- no database
- no team workspace
- no billing
- no browser automation
- no local credential storage

## Positioning

Rikka Studio is not a general chatbot. It is a small prompt workspace focused on repeatable experiments and clean notes.
